# Листинги кода для отчёта

В этой папке — **9 готовых .tex-файлов**, по одному на каждый листинг.
Каждый файл компилируется самостоятельно в Overleaf и даёт PDF с одним
куском кода.

## Файлы

| № | Файл | Что внутри |
|---|------|------------|
| 1 | `01_main.tex` | `src/main.cpp` — точка входа, фиксация локали |
| 2 | `02_parse_expression.tex` | `parse_expression` — приоритет `+ −` |
| 3 | `03_parse_term.tex` | `parse_term` — приоритет `× ÷`, деление на 0 |
| 4 | `04_parse_factor.tex` | `parse_factor` — диспетчер: число, скобки, минус, имя |
| 5 | `05_apply_function.tex` | `apply_function` — ln, exp, sqrt, abs |
| 6 | `06_parse_number.tex` | `parse_number` — локаль-независимый разбор числа |
| 7 | `07_compute.tex` | `MainWindow::compute` — главный сценарий |
| 8 | `08_keypress.tex` | `keyPressEvent` — поддержка клавиатуры |
| 9 | `09_format_result.tex` | `format_result` — вывод дробных чисел |

`_preamble.tex` — общая преамбула, подключается через `\input{_preamble}`
в каждом файле. Содержит настройки `listings`, палитру подсветки и
поддержку кириллицы.

## Как использовать в Overleaf

### Вариант A — каждый листинг отдельным проектом

1. Создай новый проект в Overleaf.
2. Загрузи в него `_preamble.tex` и нужный `NN_*.tex`.
3. Сделай `NN_*.tex` главным файлом (правый клик → Set as Main).
4. Скомпилируй — получишь PDF с одним листингом.
5. Скриншот / экспорт в картинку — вставляй в отчёт.

### Вариант B — в один проект отчёта (рекомендую)

1. Скопируй всю папку `listings/` в свой Overleaf-проект отчёта.
2. **В свою преамбулу `main.tex`** добавь содержимое
   `_preamble.tex` начиная со строки `\usepackage{listings}` (без
   `\documentclass` и `\begin{document}` — они у тебя уже есть).
3. В нужном месте отчёта подключай блок кода так:

```latex
\lstinputlisting[
    caption={Функция parse\_expression --- приоритет + и -},
    label={lst:expr}
]{listings/raw/parse_expression.cpp}
```

Это удобнее, потому что:
- листинг автоматически обновится при правке кода в проекте,
- нумерация листингов в отчёте сквозная.

Но для этого нужны «чистые» исходники без `\input{_preamble}` и
`\begin{lstlisting}`-обёртки. Если выберешь этот вариант — скажи, я
сделаю папку `listings/raw/` с голыми `.cpp`-фрагментами.

## Что компилировать

В Overleaf при открытии любого `NN_*.tex` достаточно нажать **Recompile**.
Никаких дополнительных настроек не нужно — `pdfLaTeX` (по умолчанию)
работает.

## Если всё ещё ругается

Самая частая причина ошибок — **попытка вставить только
`\begin{lstlisting}`** без преамбулы. В этой папке файлы уже содержат
всё нужное (документкласс, `\usepackage{listings}`, кириллицу,
`\begin{document}`), просто компилируй файл целиком.
