Проектная деятельность, группа М3О-103БВ-25.
 
Задание: разработать программу с графическим интерфейсом на языке C/C++ для расчёта простейших математических выражений.
 
Цель: смоделировать совместную работу студентов над проектом с распределением обязанностей по каждому отдельному компоненту разработки. Освоить работу с обработкой математических выражений и графическими библиотеками C/C++.
 
Описание: программа представляет собой калькулятор, в котором есть поле ввода математического выражения с помощью кнопок интерфейса или с клавиатуры и поле с результатом. Таким образом, задание состоит из 2 частей:
1. Разработать графический интерфейс программы с помощью сторонних библиотек C/C++ (например, GTK), который будет содержать в себе стандартные элементы типичного калькулятора:
1.1. Поле ввода выражения с возможностью стереть ненужные значения
1.2. Поле результата
1.3. Цифры: 1, 2, 3, 4, 5, 6, 7, 8, 9, 0
1.4. Стандартные арифметические операции (/, *, +, -)
1.5. Сброс (С), равно (=) и точка для дробного числа (.)
1.6. Дополнительное задание по варианту.
2. Написать простейший парсер (обработчик) математического выражения, которое пользователь вводит с помощью кнопок графического интерфейса. Данные от парсера используются в функции, которая производит сам расчёт математического выражения. Учесть возможный ошибочный ввод (например, «++» вместо «+», деление на 0, отсутствие операций и др.). Должен быть реализован учёт скобок для управления порядком вычисления.
Содержание отчёта:
Отчёт должен содержать следующие разделы:
• Титульный лист,
• Формулировка задания с указанием варианта,
• Реализация программы:
• Проектирование – описание того, из каких компонентов будет состоять программа: используемые библиотеки, описание компонентов графического интерфейса,
• Разработка – описание алгоритма работы программы (какие действия выполняют функции, как они взаимодействует с графическим интерфейсом, друг с другом), блок-схема функций алгоритма,
• Тестирование – описание тестирования разработанной программы, примеры входных и выходных данных со скриншотами.
• Заключение – краткое описание того, что было разработано и какие функции выполняет.
