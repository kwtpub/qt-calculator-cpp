## Сборка на macOS

```bash
# 1. Установить зависимости
brew install qt cmake

# 2. Собрать
cd project-1
cmake -B build -S .
cmake --build build -j

# 3. Запустить
./build/calculator.app/Contents/MacOS/calculator
# или
open ./build/calculator.app
```

## Сборка на Linux

```bash
sudo apt install qt6-base-dev cmake build-essential
cmake -B build -S .
cmake --build build -j
./build/calculator
```

## Запуск тестов парсера

```bash
cd build && ctest --output-on-failure
```

## Поддерживаемые операции

- Арифметика: `+`, `-`, `*`, `/`
- Скобки: `(`, `)`
- Унарный минус: `-5`, `-(2+3)`
- Десятичные дроби: `3.14`
- Функции (вариант 2): `ln(x)`, `exp(x)`, `sqrt(x)`, `abs(x)`

## Обработка ошибок

- Деление на ноль
- `ln(x)` при `x <= 0`
- `sqrt(x)` при `x < 0`
- Несбалансированные скобки
- Двойные операторы (`++`, `**`)
- Пустое выражение
